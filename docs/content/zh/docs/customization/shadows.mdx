# Shadows

Shadows 是预设的阴影样式集合，统一管理 UI 元素的阴影效果。

<Callout type='info'>
  <span className='x:float-left'>
    默认情况下，Shadows 类型的 Tokens 可直接应用于以下属性&nbsp;
  </span>
  <AvailableProps.Shadows />
  <span>。</span>
</Callout>

<FlexCenter gap='8' sx={{ bg: 'rgb(250, 250, 250)', p: '8', color: 'black' }}>
  {['md', 'lg', 'xl'].map((value) => (
    <Box
      key={value}
      sx={{
        boxShadow: value,
        w: 100,
        h: 100,
        borderRadius: 'sm',
        display: 'flex',
        justifyContent: 'center',
        alignItems: 'center',
      }}
    >
      {value}
    </Box>
  ))}
</FlexCenter>

Nex UI 开箱即支持以下 Tokens。

<TokenTable
  dataSource={map(defaultConfig.tokens.shadows, (value, key) => ({
    token: `shadows.${key}`,
    value,
  }))}
/>
