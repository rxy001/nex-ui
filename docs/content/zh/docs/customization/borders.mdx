# Borders

Borders Tokens 是预设的边框样式的集合，统一管理 UI 元素的边框宽度、颜色和样式。

<Callout type='info'>
  <span className='x:float-left'>
    默认情况下，Borders 类型的 Tokens 可直接应用于以下属性&nbsp;
  </span>
  <AvailableProps.Borders />
  <span>。</span>
</Callout>

Nex UI 开箱即支持以下 Tokens。

<TokensTable
  dataSource={map(defaultConfig.tokens.borders, (value, key) => ({
    token: `borders.${key}`,
    value,
  }))}
/>
