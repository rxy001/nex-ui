# Z-Indices

Z-Index Tokens 是预设的层级数值集合，确保全局层叠顺序可控且一致。

<Callout type='info'>
  <span className='x:float-left'>
    默认情况下，Z-Index Tokens 可直接应用于以下属性&nbsp;
  </span>
  <AvailableProps.ZIndices />
  <span>。</span>
</Callout>

Nex UI 开箱即支持以下 Tokens。

<TokensTable
  dataSource={map(defaultConfig.tokens.zIndices, (value, key) => ({
    token: `zIndices.${key}`,
    value,
  }))}
/>
