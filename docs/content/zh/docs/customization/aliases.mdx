# Aliases

Aliases 是用于定义 CSS 属性的别名。

## 自定义 Aliases

Aliases 可在您的主题配置 `theme.aliases` 部分定义：

```tsx
import { defineTheme } from '@nex-ui/react'

const theme = defineTheme({
  aliases: {
    w: 'width',
    h: 'height',
    px: ['paddingLeft', 'paddingRight'],
  },
})
```

Alias 值的类型为 `CSSProperty | CSSProperty[]`。

### TypeScript

之后，如果您正在使用 TypeScript，可通过 [Module Augmentation](https://www.typescriptlang.org/docs/handbook/declaration-merging.html#module-augmentation)
扩展新定义的 Aliases。

```tsx
declare module '@nex-ui/react' {
  interface AliasesOverrides {
    w: 'width'
    h: 'height'
    px: ['paddingLeft', 'paddingRight']
  }
}
```

## 使用 Aliases

在定义过后，您可以通过 [`sx`](/a) 属性、 [`defineRecipe`](/a) 或者 [`defineSlotRecipe`](/a) 使用新定义的 Aliases。

```tsx
import { Box } from '@nex-ui/react'

export default function App() {
  return (
    <Box
      sx={{
        w: '16',
        h: '16',
        px: '16',
      }}
    >
      Box
    </Box>
  )
}
```

<Callout>
  Nex UI 已关联 `width`、`height` 与 `sizes`，`paddingLeft`、`paddingRight` 与
  `spacing` 之间的映射关系，Aliases 会在内部转化为相应的 CSS
  Property，因此可以直接使用。在 [`Scales`页面]()了解更多信息。
</Callout>
