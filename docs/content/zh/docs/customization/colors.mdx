# Colors

Nex UI 开箱即提供一套默认的调色板，非常适合当您还未确定具体的品牌配色方案时使用。

这些颜色被分为‌通用型颜色和语义化颜色。

- 通用型颜色：跨主题一致性。
- 语义化主题：根据所选的主题进行调整。

<Callout type='info'>
  <span className='x:float-left'>
    默认情况下，Colors 类型的 Tokens 可直接应用于以下属性&nbsp;
  </span>
  <AvailableProps.Colors />
  <span>。</span>
</Callout>
