# Spaces

Space Tokens 是一组预设的间距值，确保 UI 元素（如外边距、内边距、间隙等）的视觉一致性。

<Callout type='info'>
  <span className='x:float-left'>
    默认情况下，Space Tokens 可直接应用于以下属性&nbsp;
  </span>
  <AvailableProps.Spaces />
  <span>。</span>
</Callout>

Nex UI 开箱即支持以下 Tokens。

export const dataSource = (() => {
  const tokens = Object.entries(defaultConfig.tokens.spaces)
    .map(([key, value]) => {
      const numValue = parseFloat(value);
      return { token:key, value: value, numValue };
    })
    .sort((a, b) => a.numValue - b.numValue) 
    .map(({ numValue, ...rest }) => rest);

return tokens.concat(tokens.map(({token,value})=>({token: `-${token}`, value: `-${value}`})))

})()

<TokensTable dataSource={dataSource} />
