# Spaces

Spaces Tokens 是一组预设的间距值，确保 UI 元素（如外边距、内边距、间隙等）的视觉一致性。

<Callout type='info'>
  <span className='x:float-left'>
    默认情况下，Spaces 类型的 Tokens 可直接应用于以下属性&nbsp;
  </span>
  <AvailableProps.Spaces />
  <span>。</span>
</Callout>

Nex UI 开箱即支持以下 Tokens。

export const dataSource = (() => {
  return Object.entries(defaultConfig.tokens.spaces)
    .map(([key, value]) => {
      const numValue = parseFloat(value);
      return { token:key, value: value, numValue };
    })
    .sort((a, b) => a.numValue - b.numValue) // 按像素值排序
    .map(({ numValue, ...rest }) => rest); // 移除临时属性

})()

<TokensTable dataSource={dataSource} />
