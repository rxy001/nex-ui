# Transitions

Transition Tokens 是预设的过渡效果的集合，确保 UI 元素不同状态变化之间的过渡效果平滑、一致。

<Callout type='info'>
  <span className='x:float-left'>
    默认情况下，Transition Tokens 可直接应用于以下属性&nbsp;
  </span>
  <AvailableProps.Transitions />
  <span>。</span>
</Callout>

Nex UI 开箱即支持以下 Tokens。

<TokensTable
  dataSource={Object.entries(defaultConfig.tokens.transitions ?? {}).map(
    ([key, value]) => ({
      token: `transitions.${key}`,
      value,
    }),
  )}
/>
