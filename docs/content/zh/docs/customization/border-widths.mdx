# BorderWidths

BorderWidth Tokens 是一组预设的边框宽度。

<Callout type='info'>
  <span className='x:float-left'>
    默认情况下，BorderWidth Tokens 可直接应用于以下属性&nbsp;
  </span>
  <AvailableProps.BorderWidths />
  <span>。</span>
</Callout>

Nex UI 开箱即支持以下 Tokens。

<TokensTable
  dataSource={Object.entries(defaultConfig.tokens.borderWidths ?? {}).map(
    ([key, value]) => ({
      token: `borderWidths.${key}`,
      value,
    }),
  )}
/>
