# BorderWidths

BorderWidths Tokens 是一组预设的边框宽度。

<Callout type='info'>
  <span className='x:float-left'>
    默认情况下，BorderWidths 类型的 Tokens 可直接应用于以下属性&nbsp;
  </span>
  <AvailableProps.BorderWidths />
  <span>。</span>
</Callout>

Nex UI 开箱即支持以下 Tokens。

<TokensTable
  dataSource={map(defaultConfig.tokens.borderWidths, (value, key) => ({
    token: `borderWidths.${key}`,
    value,
  }))}
/>
