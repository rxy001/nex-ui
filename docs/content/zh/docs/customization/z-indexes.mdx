# Z-Indexes

Z-Indexes Tokens 是预设的层级数值集合，确保全局层叠顺序可控且一致。

<Callout type='info'>
  <span className='x:float-left'>
    默认情况下，ZIndexes 类型的 Tokens 可直接应用于以下属性&nbsp;
  </span>
  <AvailableProps.ZIndexes />
  <span>。</span>
</Callout>

Nex UI 开箱即支持以下 Tokens。

<TokensTable
  dataSource={map(defaultConfig.tokens.zIndexes, (value, key) => ({
    token: `zIndexes.${key}`,
    value,
  }))}
/>
