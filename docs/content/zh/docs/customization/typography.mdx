# Typography

Typography Tokens 管理和规范排版样式，包括[`FontSizes`](#fontsizes)、[`FontFamilies`](#fontfamilies)、[`FontWeights`](#fontweights)、[`LineHeights`](#lineheights)等。确保文本在整个应用中具有统一的视觉风格和层次感。

## FontSizes

FontSizes Tokens 是一组预设的字体大小。

<Callout type='info'>
  <span className='x:float-left'>
    默认情况下，FontSizes 类型的 Tokens 可直接应用于以下属性&nbsp;
  </span>
  <AvailableProps.FontSizes />
  <span>。</span>
</Callout>

<TokensTable
  dataSource={map(defaultConfig.tokens.fontSizes, (value, key) => ({
    token: `fontSizes.${key}`,
    value,
  }))}
/>
