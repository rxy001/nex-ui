# Card

Card 包含关于单一主题的内容和操作选项。

<ComponentLinks component='card' />

---

## Usage

<Example {...card.usage} />

### Composition

您可以在 Card 中使用其他 NexUI 组件来组合成更复杂的卡片。

<Example {...card.composition} />

### With Image

<Example {...card.withImage} />

### Primary Action

Card 允许用户点击整个区域来触发主要操作。Card 的操作区域可以通过将其内容包装在 CardActionArea 组件中来指定。

<Example {...card.primaryAction} />

### Hover Effect

设置 `hoverable` 属性为 `true` 可以启用悬停效果，使卡片在鼠标悬停时具有视觉反馈。

<Example {...card.hoverEffect} />

### Blurred Card

设置 `blurred` 属性为 `true` 使卡片具有模糊效果。

<Example {...card.blurredCard} />
