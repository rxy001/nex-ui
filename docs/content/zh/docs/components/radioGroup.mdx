# RadioGroup

RadioGroup 允许用户从一组中选择一个项目。

<ComponentLinks component='radioGroup' ariaCompatible />

## Usage

<Example {...radioGroup.usage} />

### Disabled

<Example {...radioGroup.disabled} />

### Colors

<Example {...radioGroup.colors} />

### Sizes

使用 `size` 属性可以设置 RadioGroup 的大小，默认为 `md`。

<Example {...radioGroup.sizes} />

### Default Value

使用 `defaultValue` 属性可以设置 RadioGroup 的默认选中项。

<Example {...radioGroup.defaultValue} />

### Vertical

使用 `orientation` 属性可以设置 RadioGroup 的排列方向，默认为 `horizontal`。

<Example {...radioGroup.vertical} />

### Controlled

使用 `value` 和 `onValueChange` 属性可以控制 Radio 的值。

<Example {...radioGroup.controlled} />
