# Avatar

头像组件用于代表用户，并显示用户的个人资料照片、姓名首字母或备用图标。

### Fallback

如果加载头像图片时出现错误，该组件将按以下顺序回退到备用选项：

- `children` 属性

- `alt` 第一个字母
