# Flex

Flex 是用于在垂直或水平方向排列元素的弹性容器组件。

<ComponentLinks component='flex' />

## 使用

<Example {...flex.useage} />

### Direction

<Example {...flex.direction} />

### Gap

`gap` 属性用于设置子元素之间的间距，可使用 Spaces Tokens 和 Semantic Tokens。

<Example {...flex.gap} />

### Justify

设置子元素在主轴上的对齐方式。

<Example {...flex.justify} />

### Align

`align` 属性用于控制子元素在交叉轴上的对齐方式。

<Example {...flex.align} />
