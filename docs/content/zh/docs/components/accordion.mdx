# Accordion

Accordion 能够在页面上展开和折叠相关的内容部分。

<ComponentLinks component='accordion' ariaCompatible />

---

## 使用

<Example {...accordion.usage} />

### Disabled

<Example {...accordion.disabled} />

### Expand Multiple Items

将 `multiple` 属性设置为 `true`，Accordion 允许同时展开多个 items。

<Example {...accordion.expandMultiple} />

### Default Expanded Keys

通过 `defaultExpandedKeys` 属性，您可以设置默认展开的 items。

<Example {...accordion.defaultExpandedKeys} />

### Variants

Accordion 组件提供了多种变体，您可以通过 `variant` 属性来设置。

#### Outlined

<Example {...accordion.outlinedVariant} />

#### Underlined

<Example {...accordion.underlinedVariant} />

### Disabled Keys

通过 `disabledKeys` 属性，您可以设置禁用的 items。

<Example {...accordion.disabledKeys} />

### Custom Indicator

通过 `indicator` 属性，您可以设置自定义的指示器。也可以通过 `hideIndicator` 属性来隐藏指示器。

<Example {...accordion.indicator} />

### Controlled

Accordion 组件支持受控模式，您可以通过 `expandedKeys` 属性来控制展开的 items。

<Example {...accordion.controlled} />
