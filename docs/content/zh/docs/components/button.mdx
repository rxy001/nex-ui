# Button

按钮使用户能够通过一次点击执行操作并做出选择。

### Loading

添加 `loading` 属性，可在按钮内部显示一个加载指示器组件。

<Example {...button.loading} />

### With Icons

通过设置 `startIcon` 或 `endIcon` 属性，向按钮添加图标。

<Example {...button.withIcons} />

### Icon Only

添加 `IconOnly` 属性，可显示一个仅图标而无文本的按钮。
