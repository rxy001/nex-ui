# Switch

Switch 用于在选中和未选中状态之间进行切换。

### With Icons

您还可以通过使用 `startIcon` 和 `endIcon` 属性，在 Switch 的开头和结尾添加图标。

<Example {...$switch.withIcons} />

### Controlled

使用 `checked` 和 `onCheckedChange` 属性控制 Switch 的选中状态。

<Example {...$switch.controlled} />
