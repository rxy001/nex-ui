# Dialog

Dialog 显示需要关注或提供额外信息的自定义内容。

<ComponentLinks component='dialog' ariaCompatible />

## 使用

<Example {...dialog.usage} />

### MaxWidths

`maxWidth` 属性可以设置对话框的最大宽度，默认为 `md`。

<Example {...dialog.maxWidths} />

### FullScreen

使用 `fullScreen` 属性可以使对话框全屏显示。

<Example {...dialog.fullScreen} />

### Placements

`placement` 属性可以设置对话框的显示位置，默认为 `top`。

<Example {...dialog.placements} />

### HideBackdrop

使用 `hideBackdrop` 属性可以隐藏对话框的背景遮罩。

<Example {...dialog.hideBackdrop} />

### PreventScroll

使用 `preventScroll` 属性可以限制对话框打开时页面滚动。

<Example {...dialog.preventScroll} />

### KeepMounted

使用 `keepMounted` 属性可以保持对话框内容在关闭时仍然挂载。

<Example {...dialog.keepMounted} />

### Custom Motion

使用 `motionProps` 属性可以自定义 `visible` / `hidden` 的动画效果。

<Example {...dialog.customMotion} />

### Overflow Scroll

使用 `scrollBehavior` 属性可以控制对话框内容的滚动行为。

- `inside`: 内容在对话框内部滚动。
- `outside`: 内容在对话框外部滚动。

<Example {...dialog.scrollBehavior} />

<Example {...dialog.controlled} />
