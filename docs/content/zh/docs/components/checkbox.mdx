# Checkbox

Checkbox 允许用户从一个集合中选择一个或多个项目。

### Indeterminate

`Indeterminate` 将使 Checkbox 处于不确定状态，覆盖它的外观并保持它的状态，无论用户交互如何，直到设置为 `false` 。

<Example {...checkbox.indeterminate} />

### Controlled

使用 `checked` 和 `onCheckedChange` Props 来控制 Checkbox 的选中状态。
