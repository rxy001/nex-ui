# Tooltip

Tooltip 在用户与元素交互时显示一条简短的信息。

### Offset

使用 `offset` 属性可以设置 Tooltip 的偏移量，包括主轴和交叉轴的偏移。设置 `false` 可禁用偏移。

<Example {...tooltip.offset} />

### Flip

使用 `flip` 属性可以更改 Tooltip 的位置，使其保持在视图中。默认值为 `{ mainAxis: true, crossAxis: true }`。设置 `false` 可禁用翻转行为。

<Example {...tooltip.flip} />

### Shift

使用 `shift` 属性可以移动 Tooltip 以保持其在视图中。 默认为 `true`。

<Example {...tooltip.shift} />

### Interactive

使用 `interactive` 属性可以使内容在交互式时保持可见。 默认为 `true`。

<Example {...tooltip.interactive} />

### Delay

使用 `openDelay` 和 `closeDelay` 属性可以设置显示和隐藏 Tooltip 的延迟时间（以毫秒为单位）。

<Example {...tooltip.delay} />

### Controlled

<Example {...tooltip.controlled} />

### Custom Motion

使用 `motionProps` 属性自定义 `visible` / `hidden` 的动画效果。

<Example {...tooltip.customMotion} />
