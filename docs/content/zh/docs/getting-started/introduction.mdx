# NexUI

NexUI 是一个开源的 React 组件库，助力您打造出美观的用户界面。精心设计的组件，开箱即可直接应用于生产环境，大大节省开发时间与精力。

NexUI 旨在简化开发流程，提供美观且适应性强的系统设计，以提升用户体验。

### 深色模式

NexUI 不仅内置一套设计精美的深色主题，还为您提供一种快速实现深色模式的便捷方案。

### 可定制主题

NexUI 基于 Emotion 构建，赋予您高度灵活的主题定制能力，以满足多样化的设计需求。无需 Less、Sass、CSS 变量，轻松定制主题。

### 更快布局样式

NexUI 所有组件均支持 `sx` 属性，并提供了灵活通用的包装组件如 `Box`、`nex`，通过 `sx` 属性可快速对其进行定制。

### 使用 TypeScript

NexUI 使用 TypeScript 编写，无缝衔接您的 TypeScript 项目。

### 设计可组合式的组件 API

NexUI 额外提供了一种编写 CSS-in-JS 的方案，通过 `Recipes` 或者 `SlotRecipes`，轻松设计组件样式的同时，自动设计出具备类型安全的可组合式组件 API。
