# Dialog

Dialog displays custom content that requires attention or provides additional information.

<ComponentLinks component='dialog' ariaCompatible />

---

## Anatomy

NexUI provides seven components for building dialogs:

- `Dialog`: The root component that manages Dialog state.
- `DialogContent`: The component that displays the Dialog content.
- `DialogHeader`: The component that displays the Dialog header.
- `DialogBody`: The component that displays the Dialog body.
- `DialogFooter`: The component that displays the Dialog footer.
- `DialogTrigger`: The component used to open the Dialog.
- `DialogClose`: The component used to close the Dialog.

```tsx
import {
  Dialog,
  DialogContent,
  DialogHeader,
  DialogBody,
  DialogFooter,
  DialogTrigger,
  DialogClose,
} from '@nex-ui/react'
```

## Usage

<Example {...dialog.usage} />

### Hide Backdrop

Set `hideBackdrop` on `DialogContent` to `true` to hide the Dialog backdrop.

<Example {...dialog.hideBackdrop} />

### Prevent Scroll

Set `preventScroll` on `DialogContent` to `true` to prevent page scrolling while the Dialog is open.

<Example {...dialog.preventScroll} />

### Keep Mounted

Set `keepMounted` on `DialogContent` to `true` to keep Dialog content mounted when closed.

<Example {...dialog.keepMounted} />

### Sizes

Use the `size` prop on `DialogContent` to set the Dialog size. Defaults to `md`.

<Example {...dialog.sizes} />

### Placements

Use the `placement` prop on `DialogContent` to set where the Dialog appears. Defaults to `top`.

<Example {...dialog.placements} />

### Custom Motion

Use the `motionProps` prop on `DialogContent` to customize content animations for `visible`/`hidden` states.

<Example {...dialog.customMotion} />

### Overflow Scroll

Use the `scroll` prop on `DialogContent` to control Dialog scrolling behavior.

- `inside`: Content scrolls inside the Dialog.
- `outside`: Content scrolls outside the Dialog.

<Example {...dialog.scrollBehavior} />

### Asynchronous Closing

`DialogClose` supports asynchronous actions. If the `onClick` handler on `children` returns a `Promise`, the Dialog closes after the `Promise` resolves or rejects.

<Example {...dialog.asynchronousClosing} />

### Prevent Close

Call `event.preventDefault()` inside the `children` `onClick` handler of `DialogClose` to prevent closing.

<Example {...dialog.preventClose} />

### Controlled

Use the `open` and `onOpenChange` props on `Dialog` to control its open state.

<Example {...dialog.controlled} />
