# RadioGroup

RadioGroup allows users to select a single option from a list of mutually exclusive options.

<ComponentLinks component='radioGroup' ariaCompatible />

## Usage

<Example {...radioGroup.usage} />

### Disabled

<Example {...radioGroup.disabled} />

### Colors

<Example {...radioGroup.colors} />

### Sizes

Use the `size` prop to set the size of the RadioGroup. Defaults to `md`.

<Example {...radioGroup.sizes} />

### Default Value

Use the `defaultValue` prop to set the default selected option of the RadioGroup.

<Example {...radioGroup.defaultValue} />

### Vertical

Use the `orientation` prop to set the layout direction of the RadioGroup. Defaults to `horizontal`.

<Example {...radioGroup.vertical} />

### Controlled

Use the `value` and `onValueChange` props to control the radio input value.

<Example {...radioGroup.controlled} />
