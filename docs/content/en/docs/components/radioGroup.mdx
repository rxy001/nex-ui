# RadioGroup

RadioGroup allows users to select one option from a set of mutually exclusive choices.

<ComponentLinks component='radioGroup' ariaCompatible />

---

## Anatomy

NexUI provides two components for building radio groups:

- `Radio`: The component used to render a radio input.
- `RadioGroup`: The component used to group multiple radios.

```tsx
import { Radio, RadioGroup } from '@nex-ui/react'
```
