# Popover

Popover is a non-modal dialog that floats around its trigger element. It is commonly used to display additional rich content on top of an element.

<ComponentLinks component='popover' />

---

## Anatomy

NexUI exports 4 Popover-related components:

- `Popover`: The root component that displays the Popover.
- `PopoverTrigger`: The component that triggers the Popover to open.
- `PopoverContent`: The component that displays Popover content.
- `PopoverClose`: The component that triggers the Popover to close.

```tsx
import {
  Popover,
  PopoverTrigger,
  PopoverContent,
  PopoverClose,
} from '@nex-ui/react'
```

### Offset

Use the `offset` prop on `PopoverContent` to set main-axis and cross-axis offsets. Set it to `false` to disable offset.

<Example {...popover.offset} />

### Flip

Use the `flip` prop on `PopoverContent` to adjust placement and keep the Popover in view. The default is `{ mainAxis: true, crossAxis: true }`. Set it to `false` to disable flipping.

<Example {...popover.flip} />

### Shift

Use the `shift` prop on `PopoverContent` to keep the Popover in view. Defaults to `true`.

<Example {...popover.shift} />

### With Form

Popover manages focus within its content, so you can safely use it with form elements. When the Popover closes, focus returns to the trigger element.

<Example {...popover.withForm} />
