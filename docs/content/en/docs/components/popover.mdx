# Popover

Popover is a non-modal dialog that floats around its disclosure. It's commonly used for displaying additional rich content on top of something.

<ComponentLinks component='popover' />

## Usage

<Example {...popover.usage} />

<Example {...popover.colors} />

### Placements

<Example {...popover.placements} />

### Offset

Use the `offset` prop to set the Popover offset, including the main-axis and cross-axis offsets. Set to `false` to disable the offset.

<Example {...popover.offset} />

### Flip

Use the `flip` prop to change the Popover placement to keep it in view. Defaults to `{ mainAxis: true, crossAxis: true }`. Set to `false` to disable the flip behavior.

<Example {...popover.flip} />

### Shift

Use the `shift` prop to shift the Popover to keep it in view. Defaults to `true`.

<Example {...popover.shift} />

### With Form

The Popover handles the focus within the Popover content. It means that you can use the Popover with form elements without any problem. The focus returns to the trigger when the Popover closes.

<Example {...popover.withForm} />

### Controlled

Popover supports controlled mode via the `open` and `onOpenChange` props.

<Example {...popover.controlled} />

### Custom Motion

Use the `motionProps` prop to customize the animation effects for `visible` / `hidden`.

<Example {...popover.customMotion} />

## API

Popover inherits the props of `as` (by default, `m.div`).
