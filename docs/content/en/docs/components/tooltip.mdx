# Tooltip

Tooltips display a brief, informative message that appears when a user interacts with an element.

### Offset

Use the `offset` prop to set the Tooltip offset, including the main-axis and cross-axis offsets. Set to `false` to disable the offset.

<Example {...tooltip.offset} />

### Flip

Use the `flip` prop to change the Tooltip placement to keep it in view. Defaults to `{ mainAxis: true, crossAxis: true }`. Set to `false` to disable the flip behavior.

<Example {...tooltip.flip} />

### Shift

Use the `shift` prop to shift the Tooltip to keep it in view. Defaults to `true`.

<Example {...tooltip.shift} />

### Interactive

Use the `interactive` prop to keep the Tooltip open when interacting with its content. Defaults to `true`.

<Example {...tooltip.interactive} />

### Delay

Use the `openDelay` and `closeDelay` props to set the delay time (in milliseconds) for showing and hiding the tooltip.

<Example {...tooltip.delay} />

### Controlled

Tooltip supports controlled mode via the `open` and `onOpenChange` props.

<Example {...tooltip.controlled} />

### Custom Motion

Use the `motionProps` prop to customize the animation for the `visible` and `hidden` states.

<Example {...tooltip.customMotion} />
