# Tooltip

Tooltip displays a short message when users interact with an element.

<ComponentLinks component='tooltip' ariaCompatible />

---

## Anatomy

```tsx
import { Tooltip } from '@nex-ui/react'
```

### Offset

Use the `offset` prop to set main-axis and cross-axis offsets. Set it to `false` to disable offset.

<Example {...tooltip.offset} />

### Flip

Use the `flip` prop to adjust Tooltip placement and keep it in view. The default is `{ mainAxis: true, crossAxis: true }`. Set it to `false` to disable flipping.

<Example {...tooltip.flip} />

### Shift

Use the `shift` prop to keep the Tooltip in view. Defaults to `true`.

<Example {...tooltip.shift} />

### Interactive

Use the `interactive` prop to keep Tooltip content visible during interaction. Defaults to `false`.

<Example {...tooltip.interactive} />

### Delay

Use `openDelay` and `closeDelay` to set the show/hide delay (in milliseconds).

<Example {...tooltip.delay} />
