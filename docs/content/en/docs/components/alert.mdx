# Alert

Alerts are temporary notifications that provide concise feedback about an action or event.

### Status

Change the `status` of the Alert by passing the `status` prop. This affects the color scheme and icon. Alert supports `error`, `success`, `warning`, and `info` statuses.

Note: The `color` prop can override the default status color.

<Example {...alert.status} />

### Custom Icon

By default, Alert displays an appropriate icon based on the `status` prop. You can override this by providing a custom icon using the `icon` prop.

<Example {...alert.customIcon} />

### Without Icon

You can hide the icon by setting the `hideIcon` prop to `true`.

<Example {...alert.hideIcon} />

### With Action

Alert supports an `action` prop for additional actions.

<Example {...alert.withAction} />

### Closable

Set the `closable` prop to `true` to display a close button. `onClose` is called when the button is clicked.

<Example {...alert.closable} />
