# Accordion

Accordion allows you to expand and collapse related content sections on the page.

<ComponentLinks component='accordion' ariaCompatible />

---

## Anatomy

NexUI provides two components for building accordions:

- `Accordion`: The root component that displays multiple AccordionItems.
- `AccordionItem`: The component that displays an expandable/collapsible content item.

```tsx
import { Accordion, AccordionItem } from '@nex-ui/react'
```
