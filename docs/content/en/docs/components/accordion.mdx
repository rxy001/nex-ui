# Accordion

Accordion allows you to expand and collapse related content sections on the page.

<ComponentLinks component='accordion' ariaCompatible />

---

## Usage

<Example {...accordion.usage} />

### Disabled

<Example {...accordion.disabled} />

### Expand Multiple Items

Set the `multiple` prop to `true` to allow expanding multiple items simultaneously.

<Example {...accordion.expandMultiple} />

### Default Expanded Keys

Use the `defaultExpandedKeys` prop to set which items are expanded by default.

<Example {...accordion.defaultExpandedKeys} />

### Variants

Accordion provides multiple visual variants through the `variant` prop.

#### Outlined

<Example {...accordion.outlinedVariant} />

#### Underlined

<Example {...accordion.underlinedVariant} />

### Disabled Keys

Control disabled items using the `disabledKeys` prop.

<Example {...accordion.disabledKeys} />

### Custom Indicator

Customize the expand/collapse indicator using the `indicator` prop. Use `hideIndicator` to completely hide the indicator.

<Example {...accordion.indicator} />

### Controlled

Accordion supports controlled mode through the `expandedKeys` and `onExpandedKeysChange` props.

<Example {...accordion.controlled} />
