# Checkbox

Checkbox allows users to select one or more items from a collection.

### Indeterminate

The `indeterminate` prop sets a Checkbox to an indeterminate state, overriding its
appearance and maintaining it until set to `false`, regardless of user interaction.

<Example {...checkbox.indeterminate} />

### Controlled

Use the `checked` and `onCheckedChange` props to control the checked state of the Checkbox.
