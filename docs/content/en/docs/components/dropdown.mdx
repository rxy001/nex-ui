# Dropdown

Dropdown is a floating layer component used to display a list of options or actions.

<ComponentLinks component='dropdown' ariaCompatible />

---

## Anatomy

NexUI provides 14 components for building flexible dropdown menus:

- `Dropdown`: The root component that manages Dropdown state.
- `DropdownTrigger`: The trigger component used to open the Dropdown.
- `DropdownContent`: The component that renders Dropdown content.
- `DropdownItem`: The component used to render a Dropdown option.
- `DropdownItemGroup`: The component used to group Dropdown options.
- `DropdownItemGroupLabel`: The label component for Dropdown option groups.
- `DropdownRadioItemGroup`: The component used to render a single-select option group.
- `DropdownRadioItem`: The component used to render a single-select option.
- `DropdownCheckboxItemGroup`: The component used to render a multi-select option group.
- `DropdownCheckboxItem`: The component used to render a multi-select option.
- `DropdownTriggerItem`: The item component used to trigger a nested Dropdown.
- `SubDropdown`: The root component for nested Dropdowns.
- `SubDropdownContent`: The content component for nested Dropdowns.
- `DropdownDivider`: The component used to separate Dropdown options.

```tsx
import {
  Dropdown,
  DropdownTrigger,
  DropdownContent,
  DropdownItem,
  DropdownItemGroup,
  DropdownItemGroupLabel,
  DropdownRadioItemGroup,
  DropdownRadioItem,
  DropdownCheckboxItemGroup,
  DropdownCheckboxItem,
  DropdownTriggerItem,
  SubDropdown,
  SubDropdownContent,
  DropdownDivider,
} from '@nex-ui/react'
```
