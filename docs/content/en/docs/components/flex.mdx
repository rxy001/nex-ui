# Flex

Flex is an elastic container component used to arrange elements vertically or horizontally.

### Direction

<Example {...flex.direction} />

### Gap

The `gap` prop sets the spacing between child elements; you can use space tokens and semantic tokens for it.

<Example {...flex.gap} />

### Justify

The `justify` prop controls the alignment of child elements on the main axis.

<Example {...flex.justify} />

### Align

The `align` prop controls the alignment of child elements on the cross-axis.
