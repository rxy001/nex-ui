# Flex

Flex is a flexible container component used to arrange elements vertically or horizontally.

<ComponentLinks component='flex' />

---

## Anatomy

```tsx
import { Flex } from '@nex-ui/react'
```

## Usage

<Example {...flex.usage} />

### Direction

<Example {...flex.direction} />

### Gap

The `gap` prop defines spacing between child elements, and supports space tokens and semantic tokens.

<Example {...flex.gap} />

### Justify

The `justify` prop controls the alignment of child elements on the main axis.

<Example {...flex.justify} />

### Align

The `align` prop controls the alignment of child elements on the cross-axis.

<Example {...flex.align} />
