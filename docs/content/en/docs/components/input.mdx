### Label Placements

The `labelPlacement` property can be set to `inside`, `outside`, `float-outside` or `float-inside` to change the position of the label.

<Example {...input.labelPlacements} />

### Clear Button

If the `clearable` property is passed, there will be a clear button at the end of the input, which is only visible when there is a value.

<Example {...input.clearable} />

### Prefix & Suffix

The `prefix` and `suffix` properties can be set to add a prefix and a suffix.

<Example {...input.icons} />
