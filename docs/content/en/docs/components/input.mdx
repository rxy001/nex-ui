### Label Placements

The `labelPlacement` prop can be set to `inside`, `outside`, `float-outside`, or `float-inside` to change the label position.

<Example {...input.labelPlacements} />

### Clear Button

If the `clearable` prop is set to `true`, a clear button appears at the end of the Input and is only visible when there is a value.

<Example {...input.clearable} />

### Prefix & Suffix

Use the `prefix` and `suffix` props to add a prefix and suffix.
