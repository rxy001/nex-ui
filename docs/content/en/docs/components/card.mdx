# Card

Cards contain content and actions about a single subject.

<ComponentLinks component='card' />

---

## Usage

<Example {...card.usage} />

### Composition

You can use other NexUI components inside the card to compose a more complex card.

<Example {...card.composition} />

### With Image

<Example {...card.withImage} />

### Primary Action

Often a card allow users to interact with the entirety of its surface to trigger its main action.
The action area of the card can be specified by wrapping its contents in a `CardActionArea` component.

<Example {...card.primaryAction} />

### Hover Effect

Set the `hoverable` property to `true` enables the hover effect, giving the card visual feedback when the mouse hovers over it.

<Example {...card.hoverEffect} />

### Blurred Card

Set the `blurred` property to `true` enables the blur effect on the card.

<Example {...card.blurredCard} />
