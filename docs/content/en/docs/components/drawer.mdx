# Drawer

Drawer is a component that slides in from the side of the screen.

<ComponentLinks component='drawer' />

---

## Anatomy

NexUI exports 7 Drawer-related components:

- `Drawer`: The root component that displays the Drawer.
- `DrawerContent`: The component that displays the content of the Drawer.
- `DrawerHeader`: The component that displays the header of the Drawer.
- `DrawerBody`: The component that displays the body content of the Drawer.
- `DrawerFooter`: The component that displays the footer content of the Drawer.
- `DrawerTrigger`: The component that triggers the opening of the Drawer.
- `DrawerClose`: The component that triggers the closing of the Drawer.

```tsx
import {
  Drawer,
  DrawerContent,
  DrawerHeader,
  DrawerBody,
  DrawerFooter,
  DrawerTrigger,
  DrawerClose,
} from '@nex-ui/react'
```

## Usage

<Example {...drawer.usage} />

### Hide Backdrop

Set `hideBackdrop` on `DrawerContent` to `true` to hide the Drawer backdrop.

<Example {...drawer.hideBackdrop} />

### Prevent Scroll

Set `preventScroll` on `DrawerContent` to `true` to prevent page scrolling while the Drawer is open.

<Example {...drawer.preventScroll} />

### Keep Mounted

Set `keepMounted` on `DrawerContent` to `true` to keep Drawer content mounted when closed.

<Example {...drawer.keepMounted} />

### Sizes

Use the `size` prop on `DrawerContent` to set the Drawer size. Defaults to `md`.

<Example {...drawer.sizes} />

### Placements

Use the `placement` prop on `DrawerContent` to set where the Drawer appears. Defaults to `right`.

<Example {...drawer.placements} />

### Custom Motion

Use the `motionProps` prop on `DrawerContent` to customize show/hide animations.

<Example {...drawer.customMotion} />

### Asynchronously Close

`DrawerClose` supports asynchronous closing. If the `children` `onClick` handler returns a `Promise`, the Drawer closes after the `Promise` resolves or rejects.

<Example {...drawer.asynchronouslyClose} />

### Prevent Close

Call `event.preventDefault()` inside the `children` `onClick` handler of `DrawerClose` to prevent closing.

<Example {...drawer.preventClose} />

### Controlled Drawer

Use the `open` and `onOpenChange` props of Drawer to control the open state of the Drawer.

<Example {...drawer.controlled} />
