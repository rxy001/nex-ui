# Avatar

Avatar displays user representations through profile images, initials, or alternative icons.

### Fallbacks

Avatar provides fallback mechanisms in this order when image loading fails:

- `children` prop content
- First letter of `alt` text

<Example {...avatar.fallback} />

## AvatarGroup

### Group Max Count

Set the `max` prop to limit the number of Avatars displayed in the AvatarGroup. It accepts a number value.

<Example {...avatar.groupMaxCount} />

### Group Total Count

Use the `total` prop to set the total number of Avatars displayed in the AvatarGroup. It accepts a number value.

<Example {...avatar.groupTotalCount} />

### Custom Spacing

Use the `spacing` prop to set the spacing between Avatars in the AvatarGroup. It accepts a number value.

<Example {...avatar.groupSpacing} />

### Custom Surplus

`renderSurplus` allows custom rendering of surplus Avatars in the AvatarGroup. Set `renderSurplus` to a function that receives the surplus count based on `children` and `max`.

<Example {...avatar.groupCustomSurplus} />
