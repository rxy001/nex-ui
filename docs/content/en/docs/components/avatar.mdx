# Avatar

Avatar represents users with profile images, initials, or fallback icons.

<ComponentLinks component='avatar' />

---

## Anatomy

NexUI exports 2 Avatar-related components:

- `Avatar`: The component that displays a single user.
- `AvatarGroup`: The component that displays a group of users.

```tsx
import { Avatar, AvatarGroup } from '@nex-ui/react'
```

### Fallbacks

Avatar provides fallback mechanisms in this order when image loading fails:

- `children` prop content
- First letter of `alt` text

<Example {...avatar.fallback} />

## AvatarGroup

### Group Max Count

Use the `max` prop to limit how many Avatars are shown in an AvatarGroup.

<Example {...avatar.groupMaxCount} />

### Group Total Count

Use the `total` prop to set the total number of Avatars in the AvatarGroup.

<Example {...avatar.groupTotalCount} />

### Custom Spacing

Use the `spacing` prop to set the spacing between Avatars in the AvatarGroup.

<Example {...avatar.groupSpacing} />

### Custom Surplus

`renderSurplus` allows custom rendering of surplus Avatars in the AvatarGroup. Set `renderSurplus` to a function that receives the surplus count based on `children` and `max`.

<Example {...avatar.groupCustomSurplus} />
