# Recipes

Recipes let you define multiple style variants for a single component.

Recipes provide a CSS-in-JS approach with better performance, improved developer experience, and strong composability. One key feature is creating multiple variant styles through a type-safe runtime API.

Before reading this section, we recommend familiarizing yourself with [The sx prop](/docs/styling/sx-prop).

**Recipes support the same features as the sx property.**

## Basic Usage

You can use the `defineRecipe` API to define a Recipe.

```tsx {4-9,16-21}
import { defineRecipe } from '@nex-ui/react'

const buttonRecipe = defineRecipe({
  base: {
    display: 'flex',
    justifyContent: 'center',
    alignItems: 'center',
    padding: '5px 10px',
  },
})

buttonRecipe()
/**
 * Result:
 *
 * {
 *   display: 'flex',
 *   justifyContent: 'center',
 *   alignItems: 'center',
 *   padding: '5px 10px',
 * }
 */
```

## Multiple Variants

The `variants` property lets you define multiple variants for a Recipe.

```tsx {9-18, 28-31, 42-45}
import { defineRecipe } from '@nex-ui/react'

const buttonRecipe = defineRecipe({
  base: {
    display: 'flex',
    justifyContent: 'center',
    alignItems: 'center',
  },
  variants: {
    color: {
      primary: { bg: 'blue.500', color: 'white' },
      secondary: { bg: 'red.500', color: 'white' },
    },
    size: {
      sm: { padding: '4', fontSize: '12px' },
      lg: { padding: '8', fontSize: '16px' },
    },
  },
})

buttonRecipe({ color: 'primary', size: 'sm' })
/**
 * Result:
 * {
 *   display: 'flex',
 *   justifyContent: 'center',
 *   alignItems: 'center',
 *   bg: 'blue.500',
 *   color: 'white',
 *   padding: '4',
 *   fontSize: '12px'
 * }
 */

buttonRecipe({ color: 'secondary', size: 'lg' })
/**
 * Result:
 * {
 *   display: 'flex',
 *   justifyContent: 'center',
 *   alignItems: 'center',
 *   bg: 'red.500',
 *   color: 'white',
 *   padding: '8',
 *   fontSize: '16px'
 * }
 */
```

## Boolean Variants

You can also create boolean variants for a Recipe. This is useful for state-like variants (for example, `disabled`).

```tsx {15-21,39-41}
import { defineRecipe } from '@nex-ui/react'

const buttonRecipe = defineRecipe({
  base: {
    display: 'flex',
    justifyContent: 'center',
    alignItems: 'center',
    padding: '5px 10px',
  },
  variants: {
    color: {
      primary: { bg: 'blue.500', color: 'white' },
      secondary: { bg: 'red.500', color: 'white' },
    },
    disabled: {
      true: {
        'pointer-events': 'none',
        cursor: 'not-allowed',
        opacity: 0.6,
      },
    },
  },
})

buttonRecipe({
  color: 'primary',
  disabled: true,
})

/**
 * Result:
 * {
 *   display: 'flex',
 *   justifyContent: 'center',
 *   alignItems: 'center',
 *   padding: '5px 10px',
 *   bg: 'blue.500',
 *   color: 'white',
 *   pointer-events: 'none',
 *   cursor: 'not-allowed',
 *   opacity: 0.6,
 * }
 */
```

## Compound Variants

The `compoundVariants` property defines variants that are applied when specific variant combinations are matched.

```tsx {23-31,46}
import { defineRecipe } from '@nex-ui/react'

const buttonRecipe = defineRecipe({
  base: {
    display: 'flex',
    justifyContent: 'center',
    alignItems: 'center',
    padding: '5px 10px',
  },
  variants: {
    color: {
      primary: { bg: 'blue.500', color: 'white' },
      secondary: { bg: 'red.500', color: 'white' },
    },
    disabled: {
      true: {
        'pointer-events': 'none',
        cursor: 'not-allowed',
        opacity: 0.6,
      },
    },
  },
  compoundVariants: [
    {
      color: 'primary',
      disabled: true,
      css: {
        bg: 'blue.300',
      },
    },
  ],
})

buttonRecipe({
  color: 'primary',
  disabled: true,
})

/**
 * Result:
 * {
 *   display: 'flex',
 *   justifyContent: 'center',
 *   alignItems: 'center',
 *   padding: '5px 10px',
 *   bg: 'blue.300',
 *   color: 'white',
 *   pointer-events: 'none',
 *   cursor: 'not-allowed',
 *   opacity: 0.6,
 * }
 */
```

<Callout type='warning'>
  Note that the compoundVariants key is an array.
</Callout>

You can also set multiple Variants at the same time.

```tsx {25,28,50,70}
import { defineRecipe } from '@nex-ui/react'

const buttonRecipe = defineRecipe({

/**

 *   alignItems: 'center',

 *   bg: 'blue.500',

 *   opacity: 0.3,

 */

buttonRecipe({
  color: 'secondary',
  disabled: true,
})

/**
 * Result:
 * {
 *   display: 'flex',
 *   justifyContent: 'center',
 *   alignItems: 'center',
 *   padding: '5px 10px',

```

## Default Variants

The `defaultVariants` property defines the default Variant values for a Recipe. This is useful when you want a certain Variant to be applied by default.

```tsx {19-22,33-36}
import { defineRecipe } from '@nex-ui/react'

const buttonRecipe = defineRecipe({
  base: {
    display: 'flex',
    justifyContent: 'center',
    alignItems: 'center',
  },
  variants: {
    color: {
      primary: { bg: 'blue.500', color: 'white' },
      secondary: { bg: 'red.500', color: 'white' },
    },
    size: {
      sm: { padding: '4', fontSize: '12px' },
      lg: { padding: '8', fontSize: '16px' },
    },
  },
  defaultVariants: {
    color: 'primary',
    size: 'sm',
  },
})

buttonRecipe()

/**
 * Result:
 * {
 *   display: 'flex',
 *   justifyContent: 'center',
 *   alignItems: 'center',
 *   bg: 'blue.500',
 *   color: 'white',
 *   padding: '4',
 *   fontSize: '12px'
 * }
 */
```

## Extends

`defineRecipe` can extend another Recipe, including its `base`, `variants`, `defaultVariants`, and `compoundVariants`. It automatically merges matching keys and provides corresponding TypeScript type support.

```tsx {25-34}
import { defineRecipe } from '@nex-ui/react'

const buttonRecipe = defineRecipe({
  base: {
    display: 'flex',
    justifyContent: 'center',
    alignItems: 'center',
  },
  variants: {
    color: {
      primary: { bg: 'blue.500', color: 'white' },
      secondary: { bg: 'red.500', color: 'white' },
    },
    size: {
      sm: { padding: '4', fontSize: '12px' },
      lg: { padding: '8', fontSize: '16px' },
    },
  },
  defaultVariants: {
    color: 'primary',
    size: 'sm',
  },
})

const extendedRecipe = defineRecipe({
  extend: buttonRecipe,
  variants: {
    color: {
      muted: {
        bg: 'yellow',
        color: 'white',
      },
    },
  },
})
```
