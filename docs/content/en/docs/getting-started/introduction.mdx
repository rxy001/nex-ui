# Nex UI

Nex UI is an open-source React component library that helps you build beautiful user interfaces. The meticulously designed components are ready for production use out of the box, significantly saving development time and effort.

Nex UI aims to streamline development while providing a beautiful, highly adaptable design system that improves user experience.

### Web Accessibility

Nex UI components follow WAI-ARIA standards and are designed with accessibility in mind, ensuring an inclusive, high-quality experience for all users.

### Dark Mode

Nex UI includes a beautifully designed built-in dark theme and also offers a convenient way to quickly add dark mode to your applications.

### Customizable Theme

Nex UI is built on Emotion and offers flexible theme customization to meet a wide range of design needs. You can customize themes easily without Less, Sass, or CSS variables.

### Faster Style Design

All Nex UI components support the `sx` prop and include flexible utility components like `Box` and `nex`. This makes styling fast and consistent.

### Using TypeScript

Nex UI is written in TypeScript and seamlessly integrates with your TypeScript projects.

### Composable Component APIs

Nex UI also provides a CSS-in-JS solution. With `Recipes` and `SlotRecipes`, you can design component styles while automatically generating type-safe, composable component APIs—making custom component extension much easier.
