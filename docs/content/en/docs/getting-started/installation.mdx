# Installation

Install Nex UI and start building applications quickly.

## Scaffolding Your First Project

<Callout type='info'>
  The CLI requires Node.js 20+. If your package manager reports a compatibility
  warning, please upgrade Node.js.
</Callout>

Using the CLI is the easiest way to start a NexUI project. Initialize a project with the following command:

## Manual Installation

In an existing project, you can install Nex UI using one of the following package managers:

<Tabs
  items={[
    <span className='x:flex x:items-center x:gap-2'>
      <NpmIcon />
      npm
    </span>,
    <span className='x:flex x:items-center x:gap-2'>
      <PnpmIcon />
      pnpm
    </span>,
    <span className='x:flex x:items-center x:gap-2'>
      <YarnIcon />
      yarn
    </span>,
  ]}
>
  {/* prettier-ignore */}
  <Tabs.Tab className='x:mt-0!'>  
    ``` bash copy  
    npm install @nex-ui/react @emotion/react  
    ```  
  </Tabs.Tab>
  {/* prettier-ignore */}
  <Tabs.Tab className='x:mt-0!'>  
    ```bash copy  
    pnpm install @nex-ui/react @emotion/react  
    ```  
  </Tabs.Tab>
  {/* prettier-ignore */}
  <Tabs.Tab className='x:mt-0!'>  
    ```bash copy  
    yarn add @nex-ui/react @emotion/react  
    ```  
  </Tabs.Tab>
</Tabs>

Note that `react` and `react-dom` are peer dependencies, so they must be installed before installing Nex UI.

```json
"peerDependencies": {
  "react": "^19.0.0",
  "react-dom": "^19.0.0"
}
```

After installation, add `NexUIProvider` to your application's entry file.

```tsx
import { NexUIProvider } from '@nex-ui/react'

function App() {
  return (
    <NexUIProvider>
      <YourApplication />
    </NexUIProvider>
  )
}
```

Then you can start using Nex UI components in your application:

```tsx
import { Button } from '@nex-ui/react'

function Home() {
  return <Button>Press me</Button>
}
```
