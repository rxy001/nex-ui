# Installation

Install Nex UI to develop applications quickly!

## Scaffolding Your First NexUI Project

<Callout type='info'>
  The CLI requires Node.js version 20+. When your package manager issues a
  warning, please upgrade your Node.js version.
</Callout>

Using the CLI is the simplest way to start a NexUI project. You can directly initialize the project with the following command:

## Manual Installation

In your project, you can install Nex UI using the following methods:

<Tabs
  items={[
    <span className='x:flex x:items-center x:gap-2'>
      <NpmIcon />
      npm
    </span>,
    <span className='x:flex x:items-center x:gap-2'>
      <PnpmIcon />
      pnpm
    </span>,
    <span className='x:flex x:items-center x:gap-2'>
      <YarnIcon />
      yarn
    </span>,
  ]}
>
  {/* prettier-ignore */}
  <Tabs.Tab className='x:mt-0!'>  
    ``` bash copy  
    npm install @nex-ui/react @emotion/react  
    ```  
  </Tabs.Tab>
  {/* prettier-ignore */}
  <Tabs.Tab className='x:mt-0!'>  
    ```bash copy  
    pnpm install @nex-ui/react @emotion/react  
    ```  
  </Tabs.Tab>
  {/* prettier-ignore */}
  <Tabs.Tab className='x:mt-0!'>  
    ```bash copy  
    yarn add @nex-ui/react @emotion/react  
    ```  
  </Tabs.Tab>
</Tabs>

Note that `react` and `react-dom` are `PeerDependencies`, meaning you should ensure they are installed before installing Nex UI.

```json
"peerDependencies": {
  "react": "^19.0.0",
  "react-dom": "^19.0.0"
}
```

After that, add `NexUIProvider` to the entry file of your application.

```tsx
import { NexUIProvider } from '@nex-ui/react'

function App() {
  return (
    <NexUIProvider>
      <YourApplication />
    </NexUIProvider>
  )
}
```

Next, you can use NexUI components in your application:

```tsx
import { Button } from '@nex-ui/react'

function Home() {
  return <Button>Press me</Button>
}
```
