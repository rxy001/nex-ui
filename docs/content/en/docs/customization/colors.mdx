# Colors

Nex UI includes a set of default color palettes, which are ideal when your project does not yet have a defined brand color scheme.

These colors are divided into common colors and semantic colors.

- Common colors: Ensure consistency across themes.
- Semantic colors: Adapt automatically based on the active theme.
