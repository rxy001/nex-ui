# Transitions

Transition tokens are a collection of preset transition effects, ensuring smooth and consistent transition effects between different states of UI elements.

<Callout type='info'>
  <span className='x:float-left'>
    By default, transition tokens can be directly applied to the following
    properties&nbsp;
  </span>
  <AvailableProps.Transitions />
  <span>.</span>
</Callout>

Nex UI supports the following tokens out of the box.

<TokensTable
  dataSource={Object.entries(defaultConfig.tokens.transitions ?? {}).map(
    ([key, value]) => ({
      token: `transitions.${key}`,
      value,
    }),
  )}
/>
