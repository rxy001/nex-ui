# Borders

Border tokens are a collection of preset border styles, which uniformly manage the border width, color, and style of UI elements.

<Callout type='info'>
  <span className='x:float-left'>
    By default, border tokens can be directly applied to the following
    properties&nbsp;
  </span>
  <AvailableProps.Borders />
  <span>.</span>
</Callout>

Nex UI supports the following tokens out of the box.

<TokensTable
  dataSource={Object.entries(defaultConfig.tokens.borders ?? {}).map(
    ([key, value]) => ({
      token: `borders.${key}`,
      value,
    }),
  )}
/>
