# Typography

Typography tokens are used to manage and standardize typography styles, including [FontSizes](#fontsizes), [FontFamilies](#fontfamilies), [FontWeights](#fontweights), [LineHeights](#lineheights), etc. They ensure a unified visual style and hierarchy of text throughout the application.

## FontSizes

FontSize tokens are a set of preset font sizes.

<Callout type='info'>
  <span className='x:float-left'>
    By default, fontSize tokens can be directly applied to the following
    properties&nbsp;
  </span>
  <AvailableProps.FontSizes />
  <span>.</span>
</Callout>

<TokensTable
  dataSource={Object.entries(defaultConfig.tokens.fontSizes ?? {}).map(
    ([key, value]) => ({
      token: `fontSizes.${key}`,
      value,
    }),
  )}
/>
