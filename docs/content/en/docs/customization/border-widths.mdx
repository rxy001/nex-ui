# BorderWidths

BorderWidth tokens are a set of preset border widths.

<Callout type='info'>
  <span className='x:float-left'>
    By default, borderWidth tokens can be directly applied to the following
    properties&nbsp;
  </span>
  <AvailableProps.BorderWidths />
  <span>.</span>
</Callout>

Nex UI supports the following tokens out of the box.

<TokensTable
  dataSource={Object.entries(defaultConfig.tokens.borderWidths ?? {}).map(
    ([key, value]) => ({
      token: `borderWidths.${key}`,
      value,
    }),
  )}
/>
