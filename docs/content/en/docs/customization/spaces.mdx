# Spaces

Space tokens are a set of preset spacing values that ensure visual consistency of UI elements (such as margins, paddings, gaps, etc.).

<Callout type='info'>
  <span className='x:float-left'>
    By default, space tokens can be directly applied to the following
    properties&nbsp;
  </span>
  <AvailableProps.Spaces />
  <span>.</span>
</Callout>

Nex UI supports the following tokens out of the box.

export const dataSource = (() => {
  return Object.entries(defaultConfig.tokens.spaces)
    .map(([key, value]) => {
      const numValue = parseFloat(value);
      return { token: key, value: value, numValue };
    })
    .sort((a, b) => a.numValue - b.numValue) // Sort by pixel value
    .map(({ numValue, ...rest }) => rest); // Remove temporary properties

})()

<TokensTable dataSource={dataSource} />
