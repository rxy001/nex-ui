# Z-Indices

Z-Index tokens are a collection of preset layer numerical values, ensuring that the global stacking order is controllable and consistent.

<Callout type='info'>
  <span className='x:float-left'>
    By default, zIndex tokens can be directly applied to the following
    properties&nbsp;
  </span>
  <AvailableProps.ZIndices />
  <span>.</span>
</Callout>

Nex UI supports the following tokens out of the box.

<TokensTable
  dataSource={Object.entries(defaultConfig.tokens.zIndices ?? {}).map(
    ([key, value]) => ({
      token: `zIndices.${key}`,
      value,
    }),
  )}
/>
