# Shadows

Shadow tokens are a collection of preset shadow styles, which uniformly manage the shadow effects of UI elements.

<Callout type='info'>
  <span className='x:float-left'>
    By default, shadow tokens can be directly applied to the following
    properties&nbsp;
  </span>
  <AvailableProps.Shadows />
  <span>.</span>
</Callout>

<FlexCenter gap='8' sx={{ bg: 'rgb(250, 250, 250)', p: '8', color: 'black' }}>
  {['md', 'lg', 'xl'].map((value) => (
    <Box
      key={value}
      sx={{
        boxShadow: value,
        w: 100,
        h: 100,
        borderRadius: 'sm',
        display: 'flex',
        justifyContent: 'center',
        alignItems: 'center',
      }}
    >
      {value}
    </Box>
  ))}
</FlexCenter>

Nex UI supports the following tokens out of the box.

<TokensTable
  dataSource={map(defaultConfig.tokens.shadows, (value, key) => ({
    token: `shadows.${key}`,
    value,
  }))}
/>
