# Sizes

Size tokens are a set of standardized size values that uniformly manage properties such as the width, height, maximum/minimum size, etc., of UI elements.

<Callout type='info'>
  <span className='x:float-left'>
    By default, size tokens can be directly applied to the following
    properties&nbsp;
  </span>
  <AvailableProps.Sizes />
  <span>.</span>
</Callout>

Nex UI supports the following tokens out of the box.

export const dataSource = (() => {
  return Object.entries(defaultConfig.tokens.sizes)
    .map(([key, value]) => {
      const numValue = parseFloat(value);
      return { token:key, value: value, numValue };
    })
    .sort((a, b) => a.numValue - b.numValue) // Sort by pixel value
    .map(({ numValue, ...rest }) => rest); // Remove temporary properties

})()

<TokensTable dataSource={dataSource} />
