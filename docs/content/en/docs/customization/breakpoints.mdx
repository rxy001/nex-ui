# Breakpoints

Breakpoints define responsive thresholds for different screen sizes.

## Customizing Breakpoints

Breakpoints can be defined in the `theme.breakpoints` section of your theme configuration:

```tsx
import { defineTheme } from '@nex-ui/react'

const theme = defineTheme({
  breakpoints: {
    sm: '640px',
    md: '768px',
    lg: '1024px',
    xl: '1280px',
  },
})
```

Breakpoint values must be strings (for example, `'640px'`).

### TypeScript

If your project uses TypeScript, you can extend custom breakpoints through [Module Augmentation](https://www.typescriptlang.org/docs/handbook/declaration-merging.html#module-augmentation).

```tsx
declare module '@nex-ui/react' {
  interface BreakpointsOverrides {
    sm: '640px'
    md: '768px'
    lg: '1024px'
    xl: '1280px'
  }
}
```
