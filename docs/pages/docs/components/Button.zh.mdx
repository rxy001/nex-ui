# Button 按钮

按钮允许用户进行操作和选择，只需轻轻点击。

### 按钮类型

按钮有四种类型: primary (default)、outline、text 和 link。

<Cards>
  <Button>Button</Button>
  <Button variant="outline">Button</Button>
  <Button variant="text">Button</Button>
  <Button variant="link">Button</Button>
</Cards>
```tsx
<Button>Button</Button>
<Button variant="outline">Button</Button>
<Button variant="text">Button</Button>
<Button variant="link">Button</Button>
```
